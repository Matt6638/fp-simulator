# 税の壁（所得税・住民税）と配偶者控除・配偶者特別控除

出典: 国税庁「令和7年度税制改正による所得税の基礎控除の見直し等について」ほか（最下部リンク）
※ 2025年（令和7年）分以降の数値。原則 令和7年12月1日施行。

## 1. 妻自身に税金がかかり始める壁

### 住民税（約100〜110万円）
- 住民税（均等割・所得割）が発生し始める目安。**非課税ラインは自治体で異なる**（給与収入で概ね100〜110万円程度）。
- 改正で基礎控除・給与所得控除が上がったため非課税ラインも上がる方向だが、自治体の条例で差があるため一律ではない。
- 影響は緩やか（超えても数千〜数万円）で「崖」ではない。

### 所得税（旧103万円 → 123万円）
- 妻自身に所得税がかかり始めるライン。
- 仕組み: **給与所得控除（最低）65万円（旧55万）＋ 基礎控除58万円（旧48万）＝ 123万円**。ここまで所得税ゼロ。
- **さらに基礎控除の上乗せ特例**：合計所得金額132万円以下の人は基礎控除が **95万円**。
  - 給与収入160万円 → 給与所得控除65万を引くと合計所得95万円（＝132万以下）→ 基礎控除95万で課税所得ゼロ。
  - つまり **給与収入160万円までは所得税が実質ゼロ** になるケースがある（俗に「160万円の壁」）。
  - 合計所得132万円以下の95万円は恒久措置。132万円超336万円以下の88万円は令和7・8年分の措置（令和9年分以後58万）。

### 基礎控除（改正後・合計所得金額別）
| 合計所得金額 | 基礎控除額 | 備考 |
|---|---|---|
| 132万円以下 | **95万円** | 恒久 |
| 132万超336万以下 | 88万円 | 令和7・8年分（令和9年以後58万） |
| 336万超489万以下 | 68万円 | 同上（令和9年以後58万） |
| 489万超655万以下 | 63万円 | 同上（令和9年以後58万） |
| 655万超2,350万以下 | 58万円 | （旧48万） |
（旧制度の基礎控除は一律48万円）

## 2. 夫の税金が軽くなる控除（配偶者控除・配偶者特別控除）

夫（納税者本人）が受ける控除。妻の年収が上がると **段階的に** 減る。**崖ではない**ので、これ自体で手取りは逆転しない。

### 配偶者控除
- 妻の **合計所得58万円以下（給与収入123万円以下）** で満額。
- 控除額（本人900万以下）：38万円（老人控除対象配偶者48万円）。
- 改正で所得要件が48万→58万に緩和（＝給与103万→123万に）。

### 配偶者特別控除（改正後・令和7年分以降）
夫本人の合計所得900万円以下の場合の控除額：

| 妻の合計所得金額 | 妻の給与年収の目安 | 控除額（夫900万以下） |
|---|---|---|
| 58万超 95万以下 | 123万超 160万以下 | **38万円（満額）** |
| 95万超100万以下 | 160万超 165万前後 | 36万円 |
| 100万超105万以下 | | 31万円 |
| 105万超110万以下 | | 26万円 |
| 110万超115万以下 | | 21万円 |
| 115万超120万以下 | | 16万円 |
| 120万超125万以下 | | 11万円 |
| 125万超130万以下 | | 6万円 |
| 130万超133万以下 | | 3万円 |
| **133万円超** | **約201.6万円超** | **0円（控除消滅）** |

- **満額38万円の上限が 給与150万→160万 に拡大**（妻の合計所得95万以下）。
- 控除が完全にゼロになるのは妻の合計所得133万超＝**給与収入 約201.6万円**（旧制度と同じく「201万円の壁」は残る）。
- 夫本人の所得が900万超950万以下、950万超1000万以下では控除額が縮小（表は各ファイル内の3列版参照）。**夫の合計所得1000万円超だと配偶者控除・配偶者特別控除は使えない。**

## 3. シミュレーション上の扱い（重要）
- 税の壁は「妻に少し税がかかる」「夫の控除が少し減る」だけで、**いずれも連続的**。手取りの逆転（働き損）は起こさない。
- したがって **手取り逆転の判定に効くのは、次ファイルの社会保険（106/130万）と家族手当** である。税はシミュレーションの精度用に足し込む位置づけ。
